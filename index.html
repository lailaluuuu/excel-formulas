SUM
EASY

What it does

Adds numbers together. That's it. You give it numbers, it gives you the total.

=SUM(number1, number2, ...)
or
=SUM(A1:A10)
Example: Monthly expenses

Your data: Rent in A1 ($1200), Food in A2 ($400), Transport in A3 ($150)

Formula: =SUM(A1:A3)

Result: $1,750

Try it yourself

Result: 175
Pro tip: You can SUM across multiple ranges. =SUM(A1:A10, C1:C10, E5) works perfectly.
AVERAGE
EASY

What it does

Calculates the mean of a set of numbers. Adds them up and divides by how many there are.

=AVERAGE(number1, number2, ...)
or
=AVERAGE(A1:A10)
Example: Test scores

Your data: Test scores in A1:A5: 85, 92, 78, 88, 95

Formula: =AVERAGE(A1:A5)

Result: 87.6

Try it yourself

Result: 87.6
Pro tip: AVERAGE ignores empty cells but counts cells with 0. Use AVERAGEA to include text (counts as 0).
IF
EASY

What it does

Makes decisions. If something is true, do this. If not, do that.

=IF(condition, value_if_true, value_if_false)
Example: Pass or fail

Scenario: Grade is in A1. Pass if ≥60, fail otherwise.

Formula: =IF(A1>=60, "Pass", "Fail")

If A1 = 75: Pass

If A1 = 45: Fail

Try it yourself

Result: Pass
Pro tip: You can nest IFs for multiple conditions: =IF(A1>=90, "A", IF(A1>=80, "B", IF(A1>=70, "C", "F")))
VLOOKUP
MEDIUM

What it does

Finds something in the first column of a table, then returns a value from another column in the same row. Like a phone book lookup.

=VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])
Example: Product prices

Table in A1:B5:

A1: Apple | B1: $1.50

A2: Banana | B2: $0.80

A3: Orange | B3: $1.20

Formula: =VLOOKUP("Banana", A1:B5, 2, FALSE)

Result: $0.80

Translation: Find "Banana" in column A, return the value from column 2 (B)

Pro tip: The last parameter (FALSE or 0) means "exact match only". Use TRUE (or 1) only if your first column is sorted and you want approximate matches.
Common mistake: VLOOKUP can only look to the RIGHT. If what you want is to the left of your lookup column, you need INDEX/MATCH instead.
XLOOKUP
MEDIUM

What it does

The modern replacement for VLOOKUP. Can search any direction, return multiple columns, and has better defaults. Only works in Excel 365/2021+.

=XLOOKUP(lookup_value, lookup_array, return_array, [if_not_found], [match_mode])
Example: Employee lookup

Employee IDs in A2:A10

Names in B2:B10

Salaries in C2:C10

Formula: =XLOOKUP(E2, A2:A10, C2:C10, "Not Found")

Translation: Find value in E2 within A2:A10, return corresponding value from C2:C10

Pro tip: Unlike VLOOKUP, XLOOKUP can look left. It searches in one column and returns from ANY other column, regardless of position.
Why it's better than VLOOKUP: Searches any direction, returns multiple columns at once, has a built-in "if not found" parameter, and automatically does exact match.
INDEX/MATCH
HARD

What it does

The power combo. INDEX returns a value from a specific position. MATCH finds that position. Together, they're more flexible than VLOOKUP.

=INDEX(return_range, MATCH(lookup_value, lookup_range, 0))
Example: Two-way lookup

Product names in A2:A10

Prices in C2:C10

Formula: =INDEX(C2:C10, MATCH("Apple", A2:A10, 0))

Translation: Find where "Apple" appears in A2:A10 (MATCH), then return that position from C2:C10 (INDEX)

Pro tip: INDEX/MATCH can look left (VLOOKUP can't), is faster on large datasets, and won't break if you insert columns.
Advanced: Use INDEX/MATCH twice for two-way lookups (like finding a value in a table by both row and column headers).
SUMIF
MEDIUM

What it does

Adds up numbers, but only the ones that meet your criteria. Like SUM with a filter.

=SUMIF(range, criteria, [sum_range])
Example: Expenses by category

Category in A1:A5: Food, Transport, Food, Entertainment, Food

Amount in B1:B5: $50, $30, $60, $40, $45

Formula: =SUMIF(A1:A5, "Food", B1:B5)

Result: $155 (adds up 50 + 60 + 45)

Try it yourself

Calculate total of numbers greater than a threshold:

Result: 95
Pro tip: You can use wildcards in criteria. =SUMIF(A1:A10, "Food*", B1:B10) catches "Food", "Food-Groceries", "Food-Restaurants", etc.
COUNTIF
EASY

What it does

Counts how many cells meet your criteria. Simple as that.

=COUNTIF(range, criteria)
Example: Attendance tracking

Status in A1:A10: Present, Absent, Present, Present, Absent, Present...

Formula: =COUNTIF(A1:A10, "Present")

Result: 7 (if 7 cells contain "Present")

Try it yourself

Result: 4
Pro tip: Use comparison operators in quotes: =COUNTIF(A1:A10, ">50") or =COUNTIF(A1:A10, "<>"&B1) to count cells not equal to B1.
SUMIFS
MEDIUM

What it does

Like SUMIF, but with multiple conditions. Sum only when ALL criteria are met.

=SUMIFS(sum_range, criteria_range1, criteria1, [criteria_range2, criteria2], ...)
Example: Sales by region and product

Regions in A2:A20

Products in B2:B20

Sales in C2:C20

Formula: =SUMIFS(C2:C20, A2:A20, "North", B2:B20, "Widget")

Result: Sum of all sales where region is "North" AND product is "Widget"

Pro tip: Note the argument order is different from SUMIF. SUMIFS puts the sum_range FIRST, then criteria pairs.
COUNTIFS
MEDIUM

What it does

Counts cells that meet multiple criteria. All conditions must be true.

=COUNTIFS(criteria_range1, criteria1, [criteria_range2, criteria2], ...)
Example: Students by grade and attendance

Grades in A2:A50

Attendance % in B2:B50

Formula: =COUNTIFS(A2:A50, ">=90", B2:B50, ">=95")

Result: Count of students with grade ≥90 AND attendance ≥95%

Pro tip: You can mix text and numeric criteria: =COUNTIFS(A:A, "Completed", B:B, ">100", C:C, "<"&TODAY())
AVERAGEIF
EASY

What it does

Calculates the average, but only for cells that meet your criteria.

=AVERAGEIF(range, criteria, [average_range])
Example: Average score by gender

Gender in A2:A20: M, F, M, F...

Scores in B2:B20

Formula: =AVERAGEIF(A2:A20, "F", B2:B20)

Result: Average score for all rows where gender is "F"

Pro tip: Use AVERAGEIFS (with an S) for multiple criteria, just like SUMIFS and COUNTIFS.
CONCATENATE / CONCAT
EASY

What it does

Joins text from multiple cells into one cell. CONCAT is the newer, cleaner version.

=CONCAT(text1, text2, ...)
or the old way:
=CONCATENATE(text1, text2, ...)
or even simpler:
=A1&" "&B1
Example: Full names

First name in A1: John

Last name in B1: Smith

Formula: =CONCAT(A1, " ", B1)

Result: John Smith

Alternative: =A1&" "&B1 does the same thing

Pro tip: The & operator is usually faster to type. Use CONCAT when you need to combine many cells or ranges.
TEXTJOIN
MEDIUM

What it does

Joins text with a delimiter between each piece. Can ignore empty cells. Way better than CONCATENATE for lists.

=TEXTJOIN(delimiter, ignore_empty, text1, [text2], ...)
Example: Email list

Emails in A1:A5: john@email.com, , sarah@email.com, , mike@email.com

Formula: =TEXTJOIN(", ", TRUE, A1:A5)

Result: john@email.com, sarah@email.com, mike@email.com

Note: Empty cells are skipped because ignore_empty is TRUE

Pro tip: Use CHAR(10) as delimiter for line breaks. Combine with Wrap Text formatting for multi-line cells.
LEFT / RIGHT / MID
EASY

What it does

Extracts part of a text string. LEFT takes from the start, RIGHT from the end, MID from the middle.

=LEFT(text, num_chars)
=RIGHT(text, num_chars)
=MID(text, start_num, num_chars)
Example: Extract area code

Phone in A1: (555) 123-4567

Formula: =MID(A1, 2, 3)

Result: 555

Translation: Start at position 2, take 3 characters

Example: File extension

Filename in A1: document.pdf

Formula: =RIGHT(A1, 3)

Result: pdf

Pro tip: Combine with FIND or SEARCH to extract dynamic-length strings, like everything before an "@" in an email.
TRIM
EASY

What it does

Removes extra spaces from text. Leaves single spaces between words, removes all leading/trailing spaces.

=TRIM(text)
Example: Clean up messy data

Text in A1: " John Smith " (multiple spaces)

Formula: =TRIM(A1)

Result: "John Smith" (single spaces, no extras)

Pro tip: Use TRIM on imported data. Copy-pasted text often has hidden spaces that break lookups and comparisons.
SUBSTITUTE
EASY

What it does

Replaces specific text within a string. Like find-and-replace in a formula.

=SUBSTITUTE(text, old_text, new_text, [instance_num])
Example: Format phone numbers

Phone in A1: 5551234567

Formula: =SUBSTITUTE(SUBSTITUTE(A1, LEFT(A1,3), "("&LEFT(A1,3)&") "), MID(A1,4,3), MID(A1,4,3)&"-")

Simpler example: Remove dashes

Text in A1: 555-123-4567

Formula: =SUBSTITUTE(A1, "-", "")

Result: 5551234567

Pro tip: Use the instance_num parameter to replace only the 2nd, 3rd, etc. occurrence of the text.
TODAY / NOW
EASY

What it does

TODAY returns today's date. NOW returns the current date and time. Both update automatically.

=TODAY()
=NOW()
Example: Days until deadline

Deadline in A1: 12/31/2024

Formula: =A1-TODAY()

Result: Number of days from today to the deadline

Example: Age calculation

Birthdate in A1: 1/15/1990

Formula: =YEAR(TODAY())-YEAR(A1)

Result: Current age in years (approximate)

Pro tip: These are volatile functions - they recalculate every time the sheet changes. For a static timestamp, use Ctrl+; (date) or Ctrl+Shift+; (time).
DATE
EASY

What it does

Creates a proper date from separate year, month, and day values.

=DATE(year, month, day)
Example: Build a date

Year in A1: 2024

Month in B1: 12

Day in C1: 25

Formula: =DATE(A1, B1, C1)

Result: 12/25/2024

Pro tip: Use DATE with YEAR, MONTH, DAY functions to manipulate dates: =DATE(YEAR(A1)+1, MONTH(A1), DAY(A1)) adds one year to the date in A1.
DATEDIF
MEDIUM

What it does

Calculates the difference between two dates in years, months, or days. It's undocumented but still works.

=DATEDIF(start_date, end_date, unit)
Units: "Y" (years), "M" (months), "D" (days), "MD" (days ignoring months/years), "YM" (months ignoring years), "YD" (days ignoring years)
Example: Employment duration

Start date in A1: 1/15/2020

End date in B1: 6/30/2024

Formula: =DATEDIF(A1, B1, "Y")&" years, "&DATEDIF(A1, B1, "YM")&" months"

Result: 4 years, 5 months

Pro tip: For exact age calculation: =DATEDIF(birthdate, TODAY(), "Y") gives age in complete years.
EOMONTH
EASY

What it does

Returns the last day of a month, X months before or after a given date.

=EOMONTH(start_date, months)
Example: End of quarter

Date in A1: 1/15/2024

Formula: =EOMONTH(A1, 2)

Result: 3/31/2024 (2 months after January, last day)

Pro tip: Use EOMONTH(date, 0) to get the last day of the current month. Add 1 to get the first day of next month.
NETWORKDAYS
MEDIUM

What it does

Calculates working days between two dates, excluding weekends and optionally holidays.

=NETWORKDAYS(start_date, end_date, [holidays])
Example: Project timeline

Start in A1: 1/2/2024

End in B1: 1/31/2024

Holidays in D1:D5: 1/15/2024, etc.

Formula: =NETWORKDAYS(A1, B1, D1:D5)

Result: Number of working days between dates, excluding weekends and holidays

Pro tip: Use NETWORKDAYS.INTL to customize which days are weekends (e.g., if your weekend is Friday-Saturday instead of Saturday-Sunday).
MAX / MIN
EASY

What it does

MAX finds the largest number. MIN finds the smallest number. Simple.

=MAX(number1, [number2], ...)
=MIN(number1, [number2], ...)
Example: Highest score

Scores in A1:A10

Formula: =MAX(A1:A10)

Result: The highest score in the range

Pro tip: Use MAXIFS and MINIFS to find the largest/smallest value that meets criteria, like =MAXIFS(prices, categories, "Electronics").
ROUND / ROUNDUP / ROUNDDOWN
EASY

What it does

Rounds numbers. ROUND uses standard rounding, ROUNDUP always rounds up, ROUNDDOWN always rounds down.

=ROUND(number, num_digits)
=ROUNDUP(number, num_digits)
=ROUNDDOWN(number, num_digits)
Example: Price rounding

Price in A1: 19.7849

Formula: =ROUND(A1, 2)

Result: 19.78

Alternative: =ROUNDUP(A1, 2) = 19.79

Alternative: =ROUNDDOWN(A1, 2) = 19.78

Pro tip: Use negative num_digits to round to tens, hundreds, etc. =ROUND(1234, -2) = 1200
ABS
EASY

What it does

Returns the absolute value of a number (removes the negative sign).

=ABS(number)
Example: Distance calculation

Difference in A1: -15

Formula: =ABS(A1)

Result: 15

Pro tip: Useful for variance calculations where you only care about magnitude, not direction.
IFERROR
EASY

What it does

Catches errors and displays a custom message instead. Keeps your spreadsheet from looking broken.

=IFERROR(value, value_if_error)
Example: Safe division

Formula without IFERROR: =A1/B1

If B1 is 0: #DIV/0! error

Formula with IFERROR: =IFERROR(A1/B1, "Cannot divide by zero")

Result: Shows your custom message instead of an error

Pro tip: Wrap VLOOKUP in IFERROR to handle missing values: =IFERROR(VLOOKUP(...), "Not Found")
ISBLANK
EASY

What it does

Checks if a cell is empty. Returns TRUE if blank, FALSE if not.

=ISBLANK(value)
Example: Conditional calculation

Formula: =IF(ISBLANK(A1), "No data", A1*2)

Translation: If A1 is blank, show "No data", otherwise double the value

Pro tip: There's also ISTEXT, ISNUMBER, ISERROR, and more IS functions for different checks.
COUNTA
EASY

What it does

Counts non-empty cells. Unlike COUNT (which only counts numbers), COUNTA counts anything.

=COUNTA(value1, [value2], ...)
Example: Response rate

Responses in A1:A100

Formula: =COUNTA(A1:A100)

Result: Number of cells that have any content (text, numbers, dates, etc.)

Pro tip: Use COUNT for numbers only, COUNTA for everything, COUNTBLANK for empty cells.
UNIQUE
MEDIUM

What it does

Returns a list of unique values from a range. Removes duplicates automatically. Excel 365/2021+ only.

=UNIQUE(array, [by_col], [exactly_once])
Example: Unique customer list

Customer names in A1:A100 (with duplicates)

Formula: =UNIQUE(A1:A100)

Result: A list of each customer name, appearing only once

Pro tip: Combine with SORT: =SORT(UNIQUE(A1:A100)) gives you a sorted list of unique values.
FILTER
HARD

What it does

Extracts rows that meet your criteria. Like a dynamic table filter in a formula. Excel 365/2021+ only.

=FILTER(array, include, [if_empty])
Example: Sales over $1000

Products in A2:A20

Sales in B2:B20

Formula: =FILTER(A2:B20, B2:B20>1000, "No sales over $1000")

Result: Shows only rows where sales > $1000

Pro tip: Combine conditions with * (AND) or + (OR): =FILTER(A:B, (C:C="North")*(D:D>100)) filters for North region AND value > 100.
SORT
MEDIUM

What it does

Sorts a range of data dynamically. The results update automatically when source data changes. Excel 365/2021+ only.

=SORT(array, [sort_index], [sort_order], [by_col])
Example: Leaderboard

Names in A2:A10

Scores in B2:B10

Formula: =SORT(A2:B10, 2, -1)

Result: Names and scores sorted by score (column 2) in descending order (-1)

Pro tip: Use 1 for ascending, -1 for descending. Combine with FILTER: =SORT(FILTER(A:B, C:C="Active"), 2, -1)
LEN
EASY

What it does

Counts the number of characters in text, including spaces.

=LEN(text)
Example: Character limit check

Text in A1: "Hello World"

Formula: =LEN(A1)

Result: 11

Pro tip: Use with IF to enforce character limits: =IF(LEN(A1)>280, "Too long", "OK") for a Twitter-style character count.
FIND / SEARCH
MEDIUM

What it does

Finds the position of text within text. FIND is case-sensitive, SEARCH is not.

=FIND(find_text, within_text, [start_num])
=SEARCH(find_text, within_text, [start_num])
Example: Extract domain from email

Email in A1: john@company.com

Formula: =MID(A1, FIND("@", A1)+1, 100)

Result: company.com

Translation: Find "@", start 1 character after it, take 100 characters (more than enough)

Pro tip: Use IFERROR with FIND to check if text exists: =IFERROR(FIND("@", A1), "Not an email")
UPPER / LOWER / PROPER
EASY

What it does

Changes text case. UPPER = ALL CAPS, LOWER = all lowercase, PROPER = Title Case.

=UPPER(text)
=LOWER(text)
=PROPER(text)
Example: Fix inconsistent names

Name in A1: "jOhN sMiTh"

Formula: =PROPER(A1)

Result: "John Smith"

Pro tip: Use UPPER for comparison to make case-insensitive matches: =IF(UPPER(A1)=UPPER(B1), "Match", "No match")

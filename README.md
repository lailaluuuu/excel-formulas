# Excel Formulas That Actually Work

A clean, no-nonsense reference tool for learning Excel formulas. Built for quick lookups and practical learning.

## What It Does

Teaches 32 essential Excel formulas with:
- Clear explanations without jargon
- Real-world examples
- Interactive playgrounds to test formulas
- Pro tips and common mistakes
- Clean, distraction-free design

## Formulas Covered

**Basic Math & Logic**
- SUM, AVERAGE, MAX/MIN, ROUND, ABS
- IF, IFERROR, ISBLANK

**Lookup Functions**
- VLOOKUP, XLOOKUP, INDEX/MATCH

**Conditional Operations**
- SUMIF/SUMIFS, COUNTIF/COUNTIFS, AVERAGEIF

**Text Functions**
- CONCATENATE/CONCAT, TEXTJOIN, LEFT/RIGHT/MID
- TRIM, SUBSTITUTE, LEN, FIND/SEARCH
- UPPER/LOWER/PROPER

**Date Functions**
- TODAY/NOW, DATE, DATEDIF
- EOMONTH, NETWORKDAYS

**Modern Array Functions** (Excel 365/2021+)
- UNIQUE, FILTER, SORT

**Counting & More**
- COUNTA

## How to Use

1. Open `index.html` in any browser
2. Use the Quick Jump navigation to find your formula
3. Read the explanation and examples
4. Try the interactive playgrounds
5. Bookmark for future reference
